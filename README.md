# # EvoSimX

Framework expérimental d’IA évolutive distribuée inspirée du vivant.

## Concept
- Populations d’agents simples
- Évolution, sélection, mutations
- Observateur externe (GPT-like)
- Exploration parallèle (A / B / C)

## Objectif
Étudier l’émergence de structures complexes à partir de règles minimales.
